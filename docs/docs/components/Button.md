# Button 按钮
### 常用的操作按钮。

## 基础用法
使用 `type` 来定义按钮的样式。
:::demo 使用基础用法
```vue
<template>
  <ui-button type="1">123</ui-button>
  <ui-button type="2">123</ui-button>
  <ui-button type="3">123</ui-button>
</template>
```
:::
## props

```ts
type BtnType =

  | "2"

```
